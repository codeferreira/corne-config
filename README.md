A highly optimized 42-key (6-column) layout featuring **Colemak-DH** with timeless home row mods, designed for programming (JS/TS, Go, Elixir) and macOS with Aerospace window management.

## ğŸ“ Learning Colemak-DH

The transition takes 2-4 weeks to reach your previous typing speed. Tips:

1. **Practice daily** - Even 15-20 minutes helps
2. **Use [Keybr.com](https://keybr.com)** - Set layout to Colemak-DH
3. **Focus on accuracy first** - Speed follows naturally
4. **Home row mods are your friend** - Less finger travel

## ğŸ“ Customization

### Adjusting Home Row Mod Timing

In the keymap file, modify these values:
- `HRM_TAPPING_TERM 280` - Increase if getting false activations
- `HRM_QUICK_TAP 175` - Decrease if misfiring on repeated keys
- `require-prior-idle-ms 150` - Increase if misfiring while typing fast

### Adding New Macros

Add to the `macros` section following the existing pattern.

### Changing Aerospace Shortcuts

Modify the `aero_layer` bindings to match your Aerospace configuration.
