## Keys That Stay the Same
These keys are in the same position in both layouts:
- **Q, W, A, Z, X, C, V** (left hand, mostly unchanged)
- **Punctuation** (comma, period, slash)

## Keys That Moved
| QWERTY | Colemak-DH | Memory Trick |
|--------|------------|--------------|
| E → F | F is where E was | "F for Finger position" |
| R → P | P is on the old R | "P for Previous R" |
| T → B | B is where T was | |
| Y → J | J replaces Y | |
| U stays | Still U | |
| I → Y | Y is where I was | |
| O → ; | ; moves to O's spot | |
| P → ' | Apostrophe goes to P | |
| S → R | R takes S's place | "R Rolling" |
| D → S | S slides over | |
| F → T | T goes to F | "T for Tap" |
| G stays | Still G | |
| H → M | M replaces H | |
| J → N | N goes to J | "N for New J" |
| K → E | E comes home to K | "E for Easy" |
| L → I | I lands on L | |
| N → K | K moves to bottom | |
| M → H | H goes below | |
| B → D | D drops to B's row | "D goes Down" |

## The ARST Home Row (Left Hand)
The most common consonants are now under your strongest fingers:
- **A** - Pinky (same as QWERTY)
- **R** - Ring finger (very common letter)
- **S** - Middle finger (super common)
- **T** - Index finger (most common consonant!)

## The NEIO Home Row (Right Hand)
Vowels clustered on the right home row:
- **N** - Index finger (common consonant)
- **E** - Middle finger (most common letter!)
- **I** - Ring finger (common vowel)
- **O** - Pinky (common vowel)

## Common Bigrams Practice
Practice these common letter pairs:
- **TH** - Index fingers roll (both on home row!)
- **HE** - Roll from H to E
- **IN** - Roll from I to N
- **ER** - Cross-hand alternation
- **AN** - Cross-hand alternation
- **RE** - Cross-hand alternation
- **ON** - Roll O to N
- **ST** - Same hand, adjacent fingers

## Week 1 Focus: Home Row
Practice typing these words using only home row:
- **ARST side:** art, rat, star, arts, tars
- **NEIO side:** ion, one, nine, noise
- **Combined:** rain, stain, train, nation, rations

## Week 2 Focus: Top Row
Add these common top row letters:
- **W, F, P, L, U, Y**
- Words: wolf, pull, full, yup, flip, fly, ply

## Week 3 Focus: Bottom Row  
Complete the layout:
- **Z, X, C, D, V, K, H**
- Words: check, dock, hack, duck, kick

|------|--------------|-------|
| 1 | 10-20 | Very slow, hunt and peck |

| 6+ | 70+ | Back to or exceeding QWERTY |

## Tips
1. **Don't look at the keys** - Trust muscle memory to develop
2. **Practice daily** - Even 15 minutes helps
3. **Use the QWERTY toggle** - Z+X+C for emergencies
4. **Focus on accuracy** - Speed comes naturally
5. **Learn home row mods** - They're your new superpower
